///////////////CONTROLS//////////////

*Keyboard/Mouse*
------------------------------------------------
Move: A/D, Left ArrowKey/Right ArrowKey
Jump/Walljump: Space
Dash: Left Shift
Attack: Left MouseButton - Direction: WASD
Parry: Right MouseButton
Instantiate Player (controlled by keyboard): Enter

*Gamepad*
------------------------------------------------
Move: Left JoyStick
Jump/Walljump: A
Dash: B
Attack: Right Bumper - Direction: Left Joystick
Parry: Left Bumper
Instantiate Player (controlled by keyboard): Start

*Developer Settings* (Only Keyboard)
------------------------------------------------
Slow Motion: T
FPS Limit: F (4 Modes: 10, 30, 70, unlimited fps)
Path Behind Player: C
Player Box Mode: B
